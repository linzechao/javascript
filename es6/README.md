## 块级作用域(let、const)
```es6
// 不再需要自执行方法(IIFE)
// 使用花括号与let关键字配合
{
    let prefect = '完美'
    console.info(prefect)
}

// do表达式
let x = {
    let y = 4;
    y -= 2;
}

// const
// 只声明，不赋值(否则报错)
```
